# Postupashki_TimeSeries_HW1

# Анализ временного ряда цен акции NVIDIA

Проект посвящён анализу и прогнозированию дневных цен акции NVIDIA (тикер `NVDA`) с использованием классических моделей экспоненциального сглаживания. Работа оформлена в виде Jupyter‑ноутбука `ts_Fazlov_Homework_1.ipynb`.

Основные цели:

1. **Гипотеза 1: влияние стационарности**

   - Сравнить модели, обученные:
     - на исходном нестационарном ряде цен;
     - на стационаризованном ряде лог‑доходностей (логарифм + первая разность) с последующим восстановлением цен.
   - Проверить, улучшает ли приведение к стационарности качество прогнозов.

2. **Гипотеза 2: влияние частоты переобучения**

   - Исследовать, как период переобучения модели (Direct vs Rolling с различными периодами `k`) влияет на точность и вычислительную сложность.
   - Проанализировать ошибки во времени и поведение моделей в периоды резких движений рынка.

Используемые модели:

- **SES** (Simple Exponential Smoothing)  
- **Holt** (Holt’s Linear Trend)  
- **Holt‑Winters** (ETS с трендом и при необходимости сезонностью)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <ссылка-на-репозиторий>
cd <папка-репозитория>
```

### 2. Создание и активация виртуального окружения

```bash
python -m venv .venv

# Linux / macOS
source .venv/bin/activate

# Windows (PowerShell)
.\.venv\Scripts\Activate.ps1
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск Jupyter Notebook

```bash
jupyter notebook
```

Откройте ноутбук
```text
ts_Fazlov_Homework_1.ipynb
```
